## 1. 前端组件

### 1.1 组件结构

```
HybridSearch.vue (混合检索主组件)
├── SearchInput              # 查询输入
├── SearchResults            # 搜索结果（含 RRF/Reranker 分数）
├── SearchModeIndicator      # 检索模式指示器（hybrid/dense_only）
└── QueryEnhancementInfo     # 🆕 查询增强信息展示

VectorSearch.vue (向量搜索组件)
├── SearchInput              # 查询输入
└── SearchResults            # 搜索结果
```

### 1.2 HybridSearch.vue

混合检索交互组件，提供：
- 查询文本输入
- 🆕 查询增强开关（enable_query_enhancement）
- 检索模式展示（hybrid / dense_only）
- RRF 分数和 Reranker 分数的结果展示
- Reranker 可用性状态指示
- 🆕 查询增强信息展示（原始查询 → 改写查询、变体查询列表）
- 🆕 分页控制（page / page_size）
- 🆕 各阶段耗时明细展示（timing）

```vue
<template>
  <div class="hybrid-search">
    <div class="search-header">
      <h3>混合检索</h3>
      <SearchModeIndicator :mode="searchMode" :reranker-available="rerankerAvailable" />
    </div>
    
    <!-- 查询输入 -->
    <div class="search-input">
      <t-textarea
        v-model="queryText"
        placeholder="输入查询文本..."
        :maxlength="2000"
        :autosize="{ minRows: 2, maxRows: 6 }"
      />
      <div class="search-options">
        <t-select v-model="selectedCollections" placeholder="选择 Collection（最多5个）" multiple>
          <t-option v-for="col in collections" :key="col.id" :value="col.id" :label="col.name" />
        </t-select>
        <t-input-number v-model="topK" :min="1" :max="100" label="Top-K" />
        <t-switch v-model="enableReranker" label="Reranker 精排" />
        <t-switch v-model="enableQueryEnhancement" label="查询增强" />
      </div>
      <t-button theme="primary" :loading="isSearching" @click="handleSearch">
        搜索
      </t-button>
    </div>
    
    <!-- 🆕 查询增强信息 -->
    <div v-if="queryEnhancement?.enabled" class="query-enhancement-info">
      <t-tag theme="success" size="small">查询增强</t-tag>
      <div class="enhancement-detail">
        <div>原始查询: {{ queryEnhancement.original_query }}</div>
        <div>改写查询: {{ queryEnhancement.rewritten_query }}</div>
        <div v-if="queryEnhancement.is_complex">
          变体查询: {{ queryEnhancement.sub_queries.join(' | ') }}
        </div>
        <div>增强耗时: {{ queryEnhancement.enhancement_time_ms }}ms</div>
      </div>
    </div>
    
    <!-- 搜索结果 -->
    <div v-if="searchResults.length" class="search-results">
      <div class="result-meta">
        <t-tag theme="primary">{{ searchMode }}</t-tag>
        <span>总耗时: {{ timing?.total_ms || executionTimeMs }}ms</span>
        <span v-if="timing">
          (增强: {{ timing.query_enhancement_ms }}ms |
           嵌入: {{ timing.embedding_ms }}ms |
           检索: {{ timing.search_ms }}ms |
           精排: {{ timing.reranker_ms }}ms)
        </span>
        <span>共 {{ totalCount }} 条</span>
      </div>
      <div v-for="(result, index) in searchResults" :key="index" class="result-card">
        <div class="result-scores">
          <span v-if="result.rrf_score">RRF: {{ result.rrf_score?.toFixed(4) }}</span>
          <span v-if="result.reranker_score">Reranker: {{ result.reranker_score?.toFixed(4) }}</span>
          <span>相似度: {{ result.similarity_percent }}</span>
        </div>
        <div class="result-text">{{ result.text_content }}</div>
        <div class="result-meta">
          <t-tag size="small">{{ result.source_collection }}</t-tag>
          <t-tag size="small" variant="outline">{{ result.source_document }}</t-tag>
          <t-tag size="small" variant="outline">chunk #{{ result.chunk_position }}</t-tag>
        </div>
      </div>
      <!-- 🆕 分页 -->
      <t-pagination
        v-model:current="page"
        v-model:pageSize="pageSize"
        :total="totalCount"
        @change="handlePageChange"
      />
    </div>
  </div>
</template>

<script setup>
import { ref, computed, onMounted } from 'vue'
import { useVectorIndexStore } from '@/stores/vectorIndexStore'

const store = useVectorIndexStore()

const queryText = ref('')
const selectedCollections = ref([])
const topK = ref(10)
const enableReranker = ref(true)
const enableQueryEnhancement = ref(true)
const page = ref(1)
const pageSize = ref(10)

const isSearching = computed(() => store.isSearching)
const searchResults = computed(() => store.searchResults)
const searchMode = computed(() => store.searchMode)
const executionTimeMs = computed(() => store.executionTimeMs)
const timing = computed(() => store.timing)
const totalCount = computed(() => store.totalCount)
const queryEnhancement = computed(() => store.queryEnhancement)
const rerankerAvailable = computed(() => store.rerankerAvailable)
const collections = computed(() => store.collections)

async function handleSearch() {
  await store.hybridSearch({
    query_text: queryText.value,
    collection_ids: selectedCollections.value.length ? selectedCollections.value : null,
    top_k: topK.value,
    enable_query_enhancement: enableQueryEnhancement.value,
    page: page.value,
    page_size: pageSize.value
  })
}

function handlePageChange() {
  handleSearch()
}

onMounted(async () => {
  await store.checkRerankerHealth()
  await store.loadCollections()
})
</script>
```

|------|------|----------|

## 2. 数据模型

|--------|------|----------|

### 2.2 Pydantic Schema 模型

#### 2.2.1 请求模型

**SearchRequest**

```python
class SearchRequest(BaseModel):
    query_text: str          # 查询文本（1-2000字符）
    index_ids: List[str]     # 目标索引ID列表（可选）
    top_k: int = 10          # 返回结果数量（1-100）
    threshold: float = 0.5   # 相似度阈值（0-1）
    metric_type: str = "cosine"  # 相似度计算方法
```

**HybridSearchRequest**

```python
class HybridSearchRequest(BaseModel):
    query_text: str                    # 查询文本（1-2000字符）
    collection_ids: List[str] = None   # 目标 Collection ID 列表（最多5个）
    top_k: int = 10                    # 最多返回结果数量（1-100）
    threshold: float = 0.5             # 相似度阈值（0-1）
    metric_type: str = "cosine"        # 相似度计算方法
    search_mode: str = "auto"          # 检索模式（auto/hybrid/dense_only）
    reranker_top_n: int = 20           # Reranker 候选集大小（10-100）
    reranker_top_k: int = None         # Reranker 最大返回数（默认使用 top_k）
    enable_query_enhancement: bool = True  # 🆕 是否启用查询增强
    page: int = 1                      # 🆕 分页页码
    page_size: int = 10                # 🆕 每页数量（1-100）
```

#### 2.2.2 响应模型

**HybridSearchResultItem**

```python
class HybridSearchResultItem(BaseModel):
    id: str                    # 结果ID
    chunk_id: str              # 文档片段ID
    text_content: str          # 文档片段完整内容
    text_summary: str          # 文本摘要（≤200字符）
    similarity_score: float    # 原始相似度分数
    similarity_percent: str    # 相似度百分比
    rrf_score: float = None    # RRF 融合分数（混合检索模式）
    reranker_score: float = None  # Reranker 精排分数
    search_mode: str           # 检索模式标签 (hybrid/dense_only)
    source_collection: str     # 来源 Collection 名称
    source_document: str       # 来源文档名称
    chunk_position: int = None # 片段位置
    metadata: dict = None      # 元数据
    rank: int                  # 结果排名
```

**SearchTiming** 🆕

```python
class SearchTiming(BaseModel):
    query_enhancement_ms: int = 0  # 查询增强耗时
```

**QueryEnhancementInfo** 🆕

```python
class QueryEnhancementInfo(BaseModel):
```

## 3. 数据流

### 3.1 混合检索数据流

```
┌───────────────────┐
│  HybridSearch.vue │
│  (用户输入查询)     │
└────────┬──────────┘
         │ handleSearch()
         ↓
┌───────────────────┐
│ vectorIndexStore  │
│ hybridSearch()    │
└────────┬──────────┘
         │ POST /search/hybrid
         ↓
┌───────────────────┐
│   API Layer       │
│ search.py         │
└────────┬──────────┘
         │
         ↓
┌───────────────────────────────────────────────────┐
│  Service Layer - search_service.hybrid_search()   │
├───────────────────────────────────────────────────┤
│ 0. 查询增强  🆕  │ → query_enhancement_service    │
│    (Query Rewrite │   .enhance_query()             │
│     + Multi-query)│   → rewritten_query            │
│                   │   + sub_queries                │
│ 1. Embedding      │ → embedding_service            │
│    (支持跨模型     │   ._embed_query()              │
│     Multi-query   │   → 各变体并行嵌入             │
│     并行嵌入)      │                                │
│ 2. BM25 encode    │ → bm25_service.encode_query()  │
│ 3. 双路召回+RRF   │ → milvus_provider              │
│    (+Multi-query  │   .hybrid_search()             │
│     并行检索合并)  │                                │
│ 4. 三层防御  🆕   │                                │
│    ├ 候选配额控制  │                                │
│    ├ 近重复去重    │                                │
│    ├ Reranker精排  │ → reranker_service.rerank()    │
│    ├ 动态阈值过滤  │                                │
│    └ 多样性控制    │                                │
│ 5. 记录查询历史   │ → search_history INSERT         │
│ 6. 附加增强信息   │ → query_enhancement info       │
└────────┬──────────────────────────────────────────┘
         │
         ↓
┌───────────────────┐
│  返回结果到前端    │
│  更新 store 状态   │
│  - searchResults   │
│  - timing          │
│  - queryEnhancement│
│  - totalCount      │
└───────────────────┘
```

### 3.2 多 Collection 联合搜索数据流 🆕

```
┌────────────────────────────┐
│  选择多个 Collection       │
│  (最多5个，可跨模型)       │
└──────────┬─────────────────┘
           │
           ↓
┌────────────────────────────┐
│  按 Embedding 模型分组     │
│  各模型并行嵌入查询文本    │
└──────────┬─────────────────┘
           │
           ↓
┌────────────────────────────┐
│  asyncio.gather()          │
│  并行在各 Collection 检索  │
│  ├─ Collection A (hybrid)  │
│  ├─ Collection B (hybrid)  │
│  └─ Collection C (dense)   │
└──────────┬─────────────────┘
           │
           ↓
┌────────────────────────────┐
│  候选配额 → 合并候选集     │
│  近重复去重 → Reranker     │
│  动态阈值 → 多样性控制     │
│  → 最终 Top-K 结果         │
└────────────────────────────┘
```
