## 1. 功能概述

### 1.1 核心功能
搜索查询功能是RAG (Retrieval-Augmented Generation) 系统的最终用户接口，负责接收用户的自然语言查询，将其转换为向量后在索引中检索相似文档，返回最相关的文本片段供后续的生成模块使用。

### 1.2 主要特性
- ✅ **语义搜索**: 基于向量相似度的语义检索
- ✅ **多索引搜索**: 支持同时搜索多个索引
- ✅ **结果合并**: 多索引结果智能合并排序
- ✅ **阈值过滤**: 支持相似度阈值过滤
- ✅ **Top-K控制**: 可配置返回结果数量
- ✅ **搜索历史**: 自动保存搜索历史
- ✅ **历史重搜**: 支持从历史记录快速重搜
- ✅ **度量类型展示**: 自动显示索引的度量类型

### 1.3 设计原则
1. **用户友好**: 简洁的搜索界面，直观的结果展示
2. **高性能**: 毫秒级搜索响应
3. **可配置**: 灵活的搜索参数配置
4. **可追溯**: 完整的搜索历史记录

## 5. 工作流程

### 5.1 搜索执行流程

```
1. 用户输入查询文本
   └─> SearchInput.vue
       └─> 支持多行输入
       └─> Ctrl+Enter 触发搜索

2. 配置搜索参数
   └─> SearchConfig.vue
       ├─> 选择目标索引（可多选）
       ├─> 设置 Top-K（1-100）
       ├─> 设置相似度阈值（0-1）
       └─> 查看度量类型（只读）

3. 执行搜索
   └─> search_service.search()
       ├─> 验证查询文本
       ├─> 获取目标索引列表
       ├─> 将查询文本向量化
       ├─> 在各索引中执行搜索
       ├─> 合并排序结果
       ├─> 应用阈值过滤
       ├─> 格式化结果
       └─> 保存搜索历史

4. 展示搜索结果
   └─> SearchResults.vue
       ├─> 显示结果总数和耗时
       ├─> 显示结果卡片列表
       └─> 支持查看详情

5. 管理搜索历史
   └─> SearchHistory.vue
       ├─> 显示历史记录列表
       ├─> 支持点击重搜
       ├─> 支持删除单条
       └─> 支持清空全部
```

### 5.2 多索引搜索流程

```
用户选择多个索引
  │
  ▼
并行在各索引中搜索
  │
  ├─ 索引A: 返回 Top-K 结果
  ├─ 索引B: 返回 Top-K 结果
  └─ 索引C: 返回 Top-K 结果
  │
  ▼
合并所有结果
  │
  ▼
按相似度分数降序排序
  │
  ▼
应用阈值过滤
  │
  ▼
截取最终 Top-K 结果
  │
  ▼
返回给前端
```

## 6. 关键特性

### 6.1 语义搜索
- **向量化**: 使用Embedding模型将查询文本转换为向量
- **相似度计算**: 在向量空间中计算查询向量与文档向量的相似度
- **度量类型**: 支持余弦相似度、欧氏距离、点积

### 6.2 搜索配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 目标索引 | 多选 | 全部 | 不选则搜索所有可用索引 |
| Top-K | 数值 | 10 | 返回结果数量，范围1-100 |
| 阈值 | 数值 | 0.5 | 相似度阈值，范围0-1 |
| 度量类型 | 只读 | - | 由索引决定，自动显示 |

### 6.3 度量类型自动显示
- **选择索引后**: 自动显示该索引的度量类型
- **多索引选择**: 如果度量类型相同则显示，不同则显示"混合"
- **未选择索引**: 显示提示文字

### 6.4 搜索历史管理
- **自动保存**: 每次搜索自动保存历史
- **最大记录数**: 50条
- **自动清理**: 超出限制时自动删除最旧记录
- **快速重搜**: 点击历史记录可快速重搜

### 6.5 结果展示
- **相似度分数**: 显示0-1的相似度分数和百分比
- **文本内容**: 显示匹配的文本块内容
- **来源信息**: 显示来源索引和文档名称
- **元数据**: 显示文本块的元数据信息

## 7. 性能指标

### 7.1 搜索响应时间

| 场景 | 索引数量 | 向量数量 | 预期耗时 |
|------|---------|---------|---------|
| 小规模 | 1 | 1,000 | <100ms |
| 中规模 | 1 | 10,000 | <200ms |
| 大规模 | 1 | 100,000 | <500ms |
| 多索引 | 3 | 10,000/索引 | <500ms |

### 7.2 支持能力

| 指标 | 能力 |
|------|------|
| 最大查询长度 | 2000字符 |
| 最大Top-K | 100 |
| 最大索引数 | 无限制 |
| 历史记录数 | 50条 |
| 搜索超时 | 30秒 |

### 7.3 相似度分数转换

| 度量类型 | 原始值域 | 转换公式 | 转换后值域 |
|---------|---------|---------|-----------|
| 余弦(IP) | [-1, 1] | (score+1)/2 | [0, 1] |
| 欧氏(L2) | [0, ∞) | 1/(1+score) | (0, 1] |
| 点积(IP) | [-1, 1] | (score+1)/2 | [0, 1] |
