## 1. 组件架构

### 1.1 组件树结构

```
Embedding.vue (主视图)
├── DocumentSelector.vue      # 文档/分块结果选择器
├── ModelSelector.vue         # Embedding模型选择器
├── EmbeddingConfiguration.vue # 向量化配置面板
├── EmbeddingPanel.vue        # 执行面板
├── EmbeddingResults.vue      # 结果展示
│   ├── EmbeddingResultCard.vue    # 结果卡片
│   └── EmbeddingResultDisplay.vue # 结果详情
└── EmbeddingHistoryList.vue  # 历史记录列表
```

### 1.2 组件职责分配

| 组件 | 职责 | 状态来源 |
|------|------|----------|
| Embedding.vue | 页面布局、组件协调 | embeddingStore |
| DocumentSelector | 选择分块结果 | props + emit |
| ModelSelector | 选择Embedding模型 | props + emit |
| EmbeddingConfiguration | 配置向量化参数 | props + emit |
| EmbeddingPanel | 执行向量化、显示进度 | embeddingStore |
| EmbeddingResults | 展示向量化结果 | embeddingStore |
| EmbeddingHistoryList | 显示历史记录 | embeddingStore |

## 2. 核心组件详解

### 2.1 Embedding.vue (主视图)

```vue
<template>
  <div class="embedding-page">
    <div class="page-header">
      <h1>文档向量化</h1>
      <p>将分块后的文档转换为向量表示</p>
    </div>
    
    <div class="page-content">
      <!-- 左侧配置区 -->
      <div class="config-panel">
        <DocumentSelector
          v-model="selectedChunkingResult"
          :chunking-results="chunkingResults"
          :loading="loadingChunks"
        />
        
        <ModelSelector
          v-model="selectedModel"
          :models="availableModels"
        />
        
        <EmbeddingConfiguration
          v-model="embeddingConfig"
        />
        
        <EmbeddingPanel
          :chunking-result="selectedChunkingResult"
          :model="selectedModel"
          :config="embeddingConfig"
          @start="handleStart"
        />
      </div>
      
      <!-- 右侧结果区 -->
      <div class="result-panel">
        <EmbeddingResults
          :result="currentResult"
          :loading="isProcessing"
        />
        
        <EmbeddingHistoryList
          :history="embeddingHistory"
          @select="handleHistorySelect"
        />
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, computed, onMounted } from 'vue'
import { useEmbeddingStore } from '@/stores/embeddingStore'

const store = useEmbeddingStore()

// 响应式数据
const selectedChunkingResult = ref(null)
const selectedModel = ref('bge-m3')
const embeddingConfig = ref({
  batchSize: 100,
  maxRetries: 3
})

// 计算属性
const chunkingResults = computed(() => store.chunkingResults)
const availableModels = computed(() => store.models)
const currentResult = computed(() => store.currentResult)
const embeddingHistory = computed(() => store.history)
const isProcessing = computed(() => store.isProcessing)

// 生命周期
onMounted(async () => {
  await store.fetchChunkingResults()
  await store.fetchModels()
  await store.fetchHistory()
})

// 方法
async function handleStart() {
  await store.embedChunkingResult(
    selectedChunkingResult.value.id,
    selectedModel.value,
    embeddingConfig.value
  )
}

function handleHistorySelect(item) {
  store.loadHistoryDetail(item.id)
}
</script>
```

### 2.2 DocumentSelector.vue

```vue
<template>
  <div class="document-selector">
    <h3>选择分块结果</h3>
    
    <t-select
      v-model="selectedId"
      :options="options"
      :loading="loading"
      placeholder="请选择已完成的分块结果"
      filterable
      clearable
      @change="handleChange"
    >
      <template #option="{ option }">
        <div class="option-content">
          <span class="doc-name">{{ option.documentName }}</span>
          <span class="chunk-count">{{ option.chunkCount }} 个分块</span>
        </div>
      </template>
    </t-select>
    
    <div v-if="selected" class="selected-info">
      <p><strong>文档名称:</strong> {{ selected.documentName }}</p>
      <p><strong>分块数量:</strong> {{ selected.chunkCount }}</p>
      <p><strong>分块方法:</strong> {{ selected.method }}</p>
      <p><strong>创建时间:</strong> {{ formatTime(selected.createdAt) }}</p>
    </div>
  </div>
</template>

<script setup>
import { ref, computed, watch } from 'vue'

const props = defineProps({
  modelValue: Object,
  chunkingResults: Array,
  loading: Boolean
})

const emit = defineEmits(['update:modelValue'])

const selectedId = ref(null)

const options = computed(() => {
  return props.chunkingResults.map(item => ({
    value: item.id,
    label: item.documentName,
    documentName: item.documentName,
    chunkCount: item.chunkCount
  }))
})

const selected = computed(() => {
  return props.chunkingResults.find(item => item.id === selectedId.value)
})

watch(selectedId, (newId) => {
  const result = props.chunkingResults.find(item => item.id === newId)
  emit('update:modelValue', result)
})

function formatTime(dateStr) {
  return new Date(dateStr).toLocaleString()
}
</script>
```

### 2.4 EmbeddingPanel.vue

```vue
<template>
  <div class="embedding-panel">
    <h3>执行向量化</h3>
    
    <!-- 预览信息 -->
    <div class="preview-info">
      <div class="info-item">
        <span class="label">待处理分块:</span>
        <span class="value">{{ chunkCount }} 个</span>
      </div>
      <div class="info-item">
        <span class="label">预计维度:</span>
        <span class="value">{{ dimension }}</span>
      </div>
      <div class="info-item">
        <span class="label">预计存储:</span>
        <span class="value">{{ estimatedSize }}</span>
      </div>
    </div>
    
    <!-- 进度条 -->
    <div v-if="isProcessing" class="progress-section">
      <t-progress
        :percentage="progress"
        :status="progressStatus"
      />
      <p class="progress-text">
        已处理 {{ processedCount }} / {{ totalCount }} 个分块
      </p>
    </div>
    
    <!-- 操作按钮 -->
    <div class="actions">
      <t-button
        theme="primary"
        :loading="isProcessing"
        :disabled="!canStart"
        @click="handleStart"
      >
        {{ isProcessing ? '处理中...' : '开始向量化' }}
      </t-button>
      
      <t-button
        v-if="isProcessing"
        theme="danger"
        variant="outline"
        @click="handleCancel"
      >
        取消
      </t-button>
    </div>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'
import { useEmbeddingStore } from '@/stores/embeddingStore'

const props = defineProps({
  chunkingResult: Object,
  model: String,
  config: Object
})

const emit = defineEmits(['start', 'cancel'])

const store = useEmbeddingStore()

const isProcessing = computed(() => store.isProcessing)
const progress = computed(() => store.progress)
const processedCount = computed(() => store.processedCount)
const totalCount = computed(() => store.totalCount)

const chunkCount = computed(() => props.chunkingResult?.chunkCount || 0)
const dimension = computed(() => {
  const modelInfo = store.models.find(m => m.name === props.model)
  return modelInfo?.dimension || 0
})

const estimatedSize = computed(() => {
  const bytes = chunkCount.value * dimension.value * 4 // float32
  if (bytes < 1024) return `${bytes} B`
  if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`
  return `${(bytes / 1024 / 1024).toFixed(1)} MB`
})

const canStart = computed(() => {
  return props.chunkingResult && props.model && !isProcessing.value
})

const progressStatus = computed(() => {
  if (progress.value === 100) return 'success'
  if (store.hasError) return 'error'
  return 'active'
})

function handleStart() {
  emit('start')
}

function handleCancel() {
  store.cancelEmbedding()
  emit('cancel')
}
</script>
```

## 3. 状态管理

### 3.1 embeddingStore.js

```javascript
// stores/embeddingStore.js
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import { embeddingApi } from '@/services/embeddingApi'

export const useEmbeddingStore = defineStore('embedding', () => {
  // 状态
  const models = ref([])
  const chunkingResults = ref([])
  const history = ref([])
  const currentResult = ref(null)
  
  // 处理状态
  const isProcessing = ref(false)
  const progress = ref(0)
  const processedCount = ref(0)
  const totalCount = ref(0)
  const hasError = ref(false)
  const errorMessage = ref('')
  
  // 加载状态
  const loadingModels = ref(false)
  const loadingChunks = ref(false)
  const loadingHistory = ref(false)
  
  // 计算属性
  const completedChunkingResults = computed(() => {
    return chunkingResults.value.filter(r => r.status === 'completed')
  })
  
  // Actions
  async function fetchModels() {
    loadingModels.value = true
    try {
      const response = await embeddingApi.getModels()
      models.value = response.data.models
    } catch (error) {
      console.error('Failed to fetch models:', error)
    } finally {
      loadingModels.value = false
    }
  }
  
  async function fetchChunkingResults() {
    loadingChunks.value = true
    try {
      const response = await embeddingApi.getChunkingResults()
      chunkingResults.value = response.data.items
    } catch (error) {
      console.error('Failed to fetch chunking results:', error)
    } finally {
      loadingChunks.value = false
    }
  }
  
  async function fetchHistory(page = 1, pageSize = 20) {
    loadingHistory.value = true
    try {
      const response = await embeddingApi.getHistory(page, pageSize)
      history.value = response.data.items
    } catch (error) {
      console.error('Failed to fetch history:', error)
    } finally {
      loadingHistory.value = false
    }
  }
  
  async function embedChunkingResult(chunkingResultId, model, config) {
    isProcessing.value = true
    hasError.value = false
    errorMessage.value = ''
    progress.value = 0
    processedCount.value = 0
    
    try {
      // 获取分块数量
      const chunkResult = chunkingResults.value.find(r => r.id === chunkingResultId)
      totalCount.value = chunkResult?.chunkCount || 0
      
      // 调用API
      const response = await embeddingApi.embedChunkingResult(
        chunkingResultId,
        model,
        config.batchSize
      )
      
      currentResult.value = response.data
      progress.value = 100
      processedCount.value = totalCount.value
      
      // 刷新历史
      await fetchHistory()
      
      return response.data
    } catch (error) {
      hasError.value = true
      errorMessage.value = error.message
      throw error
    } finally {
      isProcessing.value = false
    }
  }
  
  async function loadHistoryDetail(id) {
    try {
      const response = await embeddingApi.getHistoryDetail(id)
      currentResult.value = response.data
    } catch (error) {
      console.error('Failed to load history detail:', error)
    }
  }
  
  function cancelEmbedding() {
    // 取消当前处理
    isProcessing.value = false
    hasError.value = true
    errorMessage.value = '用户取消'
  }
  
  function reset() {
    currentResult.value = null
    isProcessing.value = false
    progress.value = 0
    processedCount.value = 0
    totalCount.value = 0
    hasError.value = false
    errorMessage.value = ''
  }
  
  return {
    // 状态
    models,
    chunkingResults,
    completedChunkingResults,
    history,
    currentResult,
    isProcessing,
    progress,
    processedCount,
    totalCount,
    hasError,
    errorMessage,
    loadingModels,
    loadingChunks,
    loadingHistory,
    
    // Actions
    fetchModels,
    fetchChunkingResults,
    fetchHistory,
    embedChunkingResult,
    loadHistoryDetail,
    cancelEmbedding,
    reset
  }
})
```

## 4. 数据流与通信

### 4.1 数据流图

```
┌─────────────────────────────────────────────────────────────┐
│                       embeddingStore                         │
├─────────────────────────────────────────────────────────────┤
│  models, chunkingResults, history, currentResult            │
│  isProcessing, progress, processedCount, totalCount         │
└─────────────────────────────────────────────────────────────┘
                    ↑                    ↓
            (fetch/update)         (computed/ref)
                    │                    │
┌───────────────────┴────────────────────┴───────────────────┐
│                     Embedding.vue                           │
├─────────────────────────────────────────────────────────────┤
│  selectedChunkingResult, selectedModel, embeddingConfig     │
└─────────────────────────────────────────────────────────────┘
         │              │              │              │
    (v-model)      (v-model)      (v-model)      (props)
         ↓              ↓              ↓              ↓
┌────────────┐  ┌────────────┐  ┌────────────┐  ┌────────────┐
│ Document   │  │   Model    │  │  Embedding │  │ Embedding  │
│ Selector   │  │  Selector  │  │   Config   │  │   Panel    │
└────────────┘  └────────────┘  └────────────┘  └────────────┘
```

### 4.2 组件通信方式

| 通信类型 | 方式 | 示例 |
|---------|------|------|
| 父→子 | props | `:models="availableModels"` |
| 子→父 | emit | `@change="handleChange"` |
| 双向绑定 | v-model | `v-model="selectedModel"` |
| 跨组件 | Pinia Store | `useEmbeddingStore()` |

## 5. 工作流程

### 5.1 页面初始化流程

```
1. Embedding.vue mounted
   │
   ├─> store.fetchChunkingResults()
   │   └─> 获取已完成的分块结果列表
   │
   ├─> store.fetchModels()
   │   └─> 获取可用的Embedding模型
   │
   └─> store.fetchHistory()
       └─> 获取向量化历史记录
```

### 5.2 向量化执行流程

```
1. 用户选择分块结果
   └─> DocumentSelector emit → Embedding.vue 更新 selectedChunkingResult

2. 用户选择模型
   └─> ModelSelector emit → Embedding.vue 更新 selectedModel

3. 用户配置参数
   └─> EmbeddingConfiguration emit → Embedding.vue 更新 embeddingConfig

4. 用户点击开始
   └─> EmbeddingPanel emit('start')
       └─> Embedding.vue handleStart()
           └─> store.embedChunkingResult()
               ├─> 设置 isProcessing = true
               ├─> 调用 API
               ├─> 更新 currentResult
               ├─> 刷新 history
               └─> 设置 isProcessing = false

5. 结果展示
   └─> EmbeddingResults 监听 store.currentResult 变化
       └─> 显示向量化结果详情
```

## 6. 最佳实践

### 6.1 组件设计原则

1. **单一职责**: 每个组件只负责一个功能
2. **Props Down, Events Up**: 数据向下传递，事件向上冒泡
3. **状态提升**: 共享状态放在最近的公共父组件或Store
