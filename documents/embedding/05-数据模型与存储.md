## 1. 数据模型概述

### 1.1 存储架构

文档向量化模块采用**混合存储**架构：

| 存储类型 | 用途 | 数据内容 |
|---------|------|----------|
| 关系数据库 | 元数据管理 | 任务信息、状态、统计 |
| JSON文件 | 向量数据 | 向量数组、分块元数据 |

### 1.2 设计原则

1. **元数据与数据分离**: 元数据存数据库，大数据存文件
2. **可追溯性**: 保留完整的处理历史
3. **高效查询**: 合理的索引设计
4. **数据完整性**: 外键约束和事务保证

## 3. 数据库模型详解

### 3.1 EmbeddingResult 模型

```python
# backend/src/models/embedding_models.py

from sqlalchemy import Column, String, Integer, DateTime, Enum, ForeignKey, Text
from sqlalchemy.orm import relationship
from datetime import datetime
import enum

class EmbeddingStatus(enum.Enum):
    """向量化状态枚举"""
    PENDING = "pending"         # 等待处理
    PROCESSING = "processing"   # 处理中
    COMPLETED = "completed"     # 完成
    FAILED = "failed"           # 失败
    PARTIAL = "partial"         # 部分成功

class EmbeddingResult(Base):
    """向量化结果模型"""
    __tablename__ = "embedding_results"
    
    # 主键
    id = Column(String(36), primary_key=True)
    
    # 关联
    chunking_result_id = Column(String(36), ForeignKey("chunking_results.id"), nullable=False)
    document_id = Column(String(36), ForeignKey("documents.id"), nullable=True)
    
    # 模型信息
    model_name = Column(String(100), nullable=False, comment="Embedding模型名称")
    dimension = Column(Integer, nullable=False, comment="向量维度")
    
    # 统计信息
    total_chunks = Column(Integer, default=0, comment="总分块数")
    success_count = Column(Integer, default=0, comment="成功数量")
    failed_count = Column(Integer, default=0, comment="失败数量")
    
    # 状态
    status = Column(Enum(EmbeddingStatus), default=EmbeddingStatus.PENDING, comment="状态")
    error_message = Column(Text, nullable=True, comment="错误信息")
    
    # 结果文件
    result_file = Column(String(500), nullable=True, comment="结果文件路径")
    
    # 时间戳
    created_at = Column(DateTime, default=datetime.utcnow, comment="创建时间")
    completed_at = Column(DateTime, nullable=True, comment="完成时间")
    
    # 关系
    chunking_result = relationship("ChunkingResult", back_populates="embedding_results")
    document = relationship("Document", back_populates="embedding_results")
    
    def __repr__(self):
        return f"<EmbeddingResult(id={self.id}, model={self.model_name}, status={self.status})>"
    
    @property
    def duration_seconds(self):
        """计算处理耗时（秒）"""
        if self.completed_at and self.created_at:
            return (self.completed_at - self.created_at).total_seconds()
        return None
    
    @property
    def success_rate(self):
        """计算成功率"""
        if self.total_chunks == 0:
            return 0
        return self.success_count / self.total_chunks * 100
```

ON embedding_results(model_name);

-- 时间排序索引
CREATE INDEX idx_embedding_results_created_at 
ON embedding_results(created_at DESC);

ON embedding_results(status, created_at DESC);

### 5.2 索引使用场景

| 索引 | 使用场景 |
|------|----------|
| idx_chunking_result_id | 根据分块结果查询向量化记录 |
| idx_document_id | 根据文档查询所有向量化记录 |
| idx_status | 筛选特定状态的记录 |
| idx_model_name | 按模型统计使用情况 |

## 6. 数据一致性

### 6.1 事务处理

```python
# 向量化完成后的事务处理
async def complete_embedding(self, result_id: str, embeddings: list):
    async with self.db.begin():
        # 1. 更新数据库记录
        result = await self.db.get(EmbeddingResult, result_id)
        result.status = EmbeddingStatus.COMPLETED
        result.success_count = len(embeddings)
        result.completed_at = datetime.utcnow()
        
        # 2. 保存JSON文件
        result_file = await self._save_result_file(result_id, embeddings)
        result.result_file = result_file
        
        # 3. 提交事务
        await self.db.commit()
```

### 6.2 错误恢复

```python
# 失败时的回滚处理
async def handle_embedding_error(self, result_id: str, error: Exception):
    async with self.db.begin():
        result = await self.db.get(EmbeddingResult, result_id)
        result.status = EmbeddingStatus.FAILED
        result.error_message = str(error)
        result.completed_at = datetime.utcnow()
        
        # 删除可能已创建的部分结果文件
        if result.result_file:
            await self._delete_result_file(result.result_file)
            result.result_file = None
        
        await self.db.commit()
```

### 6.3 数据清理策略

```python
# 定期清理过期数据
async def cleanup_old_results(self, days: int = 30):
    """清理指定天数前的失败记录"""
    cutoff_date = datetime.utcnow() - timedelta(days=days)
    
    async with self.db.begin():
        # 查询过期的失败记录
        old_results = await self.db.query(EmbeddingResult).filter(
            EmbeddingResult.status == EmbeddingStatus.FAILED,
            EmbeddingResult.created_at < cutoff_date
        ).all()
        
        for result in old_results:
            # 删除关联文件
            if result.result_file:
                await self._delete_result_file(result.result_file)
            
            # 删除数据库记录
            await self.db.delete(result)
        
        await self.db.commit()
        
        return len(old_results)
```
