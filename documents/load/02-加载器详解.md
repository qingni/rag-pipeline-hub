## 1. 加载器概览

### 1.1 加载器对比表

| 加载器 | 支持格式 | 优点 | 缺点 | 适用场景 | 性能 |
|-------|---------|------|------|---------|------|
| **PyMuPDF** | PDF | 快速、准确、功能全 | 依赖C库 | 一般PDF文档 | ⭐⭐⭐⭐⭐ |
| **PyPDF** | PDF | 纯Python、轻量 | 功能有限 | 简单PDF | ⭐⭐⭐⭐ |
| **Unstructured** | PDF等 | 支持复杂布局 | 速度慢 | 复杂文档 | ⭐⭐⭐ |
| **DOCX** | DOCX | 官方支持、准确 | 仅DOCX | Word文档 | ⭐⭐⭐⭐⭐ |
| **DOC** | DOC | 支持旧格式 | 需外部工具 | 旧版Word | ⭐⭐⭐ |
| **Text** | TXT/MD | 极快、无依赖 | 仅纯文本 | 文本文件 | ⭐⭐⭐⭐⭐ |

## 2. PDF加载器

### 2.1 PyMuPDF Loader (默认)

#### 2.1.1 基本信息
- **依赖**: `PyMuPDF` (fitz)
- **版本**: 1.23.8+
- **性能**: ⭐⭐⭐⭐⭐
- **推荐度**: ⭐⭐⭐⭐⭐

#### 2.1.2 核心功能

```python
class PyMuPDFLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取PDF文本内容"""
        doc = fitz.open(file_path)
        
        # 提取元数据
        metadata = {
            "page_count": len(doc),
            "title": doc.metadata.get("title", ""),
            "author": doc.metadata.get("author", ""),
            "subject": doc.metadata.get("subject", ""),
            "creator": doc.metadata.get("creator", ""),
            "producer": doc.metadata.get("producer", ""),
            "format": doc.metadata.get("format", "")
        }
        
        # 逐页提取文本
        pages = []
        for page_num in range(len(doc)):
            page = doc[page_num]
            text = page.get_text()
            
            pages.append({
                "page_number": page_num + 1,
                "text": text,
                "char_count": len(text)
            })
        
        return {
            "success": True,
            "loader": "pymupdf",
            "metadata": metadata,
            "pages": pages,
            "full_text": "\n\n".join([p["text"] for p in pages]),
            "total_pages": len(pages),
            "total_chars": sum(p["char_count"] for p in pages)
        }
```

    def extract_text(self, file_path: str) -> Dict[str, Any]:

### 2.3 Unstructured Loader (高级)

#### 2.3.1 基本信息
- **依赖**: `unstructured`
- **版本**: 0.11.0+
- **性能**: ⭐⭐⭐
- **推荐度**: ⭐⭐⭐⭐ (复杂文档)

#### 2.3.2 核心功能

```python
class UnstructuredLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """使用Unstructured提取文本"""
        from unstructured.partition.auto import partition
        
        elements = partition(filename=file_path)
        
        # 组织为页面结构
        pages = []
        current_page = {"page_number": 1, "text": "", "char_count": 0}
        
        for element in elements:
            text = str(element)
            current_page["text"] += text + "\n"
            current_page["char_count"] += len(text)
            
            # 简单分页逻辑(可根据element类型优化)
            if len(current_page["text"]) > 2000:
                pages.append(current_page)
                current_page = {
                    "page_number": len(pages) + 1,
                    "text": "",
                    "char_count": 0
                }
        
        if current_page["text"]:
            pages.append(current_page)
        
        return {
            "success": True,
            "loader": "unstructured",
            "pages": pages,
            "total_pages": len(pages),
            "total_chars": sum(p["char_count"] for p in pages)
        }
```

#### 2.3.3 使用场景

✅ **适合**:
- 复杂文档结构
- 需要识别表格
- 多列布局
- 学术论文

❌ **不适合**:
- 简单文档(性能浪费)
- 实时处理
- 资源受限环境

## 3. Office文档加载器

### 3.1 DOCX Loader

#### 3.1.1 基本信息
- **依赖**: `python-docx`
- **版本**: 1.1.0+
- **性能**: ⭐⭐⭐⭐⭐
- **推荐度**: ⭐⭐⭐⭐⭐

#### 3.1.2 核心功能

```python
class DocxLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取DOCX文本"""
        from docx import Document
        
        doc = Document(file_path)
        
        # 提取段落
        paragraphs = []
        full_text = []
        
        for para in doc.paragraphs:
            if para.text.strip():
                paragraphs.append(para.text)
                full_text.append(para.text)
        
        # 组织为"页面"(每5段一页)
        pages = []
        page_size = 5
        for i in range(0, len(paragraphs), page_size):
            page_paras = paragraphs[i:i+page_size]
            page_text = "\n\n".join(page_paras)
            pages.append({
                "page_number": len(pages) + 1,
                "text": page_text,
                "char_count": len(page_text)
            })
        
        return {
            "success": True,
            "loader": "docx",
            "pages": pages,
            "full_text": "\n\n".join(full_text),
            "total_pages": len(pages),
            "total_chars": sum(p["char_count"] for p in pages)
        }
```

#### 3.1.3 使用场景

✅ **适合**:
- Microsoft Word文档
- 现代Office格式
- 包含格式化文本
- 生产环境

#### 3.1.4 扩展功能

**表格提取**:

def extract_tables(self, file_path: str) -> List[List[List[str]]]:
    """提取表格数据"""
    doc = Document(file_path)
    tables = []
    
    for table in doc.tables:
        table_data = []
        for row in table.rows:
            row_data = [cell.text for cell in row.cells]
            table_data.append(row_data)
        tables.append(table_data)
    
    return tables

#### 3.2.2 多方法支持

```python
class DocLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """尝试多种方法提取DOC文本"""
        
        # 方法1: antiword (最准确)
        text = self._extract_with_antiword(file_path)
        if text:
            return self._format_result(text, "antiword")
        
        # 方法2: python-docx (部分支持)
        text = self._extract_with_docx(file_path)
        if text:
            return self._format_result(text, "docx-fallback")
        
        # 方法3: textract (最后手段)
        text = self._extract_with_textract(file_path)
        if text:
            return self._format_result(text, "textract")
        
        return {
            "success": False,
            "loader": "doc",
            "error": "无法提取DOC文件内容"
        }
    
    def _extract_with_antiword(self, file_path: str) -> str:
        """使用antiword命令行工具"""
        try:
            import subprocess
            result = subprocess.run(
                ['antiword', file_path],
                capture_output=True,
                text=True,
                timeout=30
            )
            return result.stdout if result.returncode == 0 else None
        except Exception:
            return None
```

- 历史文档迁移

❌ **不适合**:
- 新建文档(应用DOCX)
- 需要保留格式

## 4. 文本加载器

### 4.1 Text Loader

#### 4.1.1 基本信息
- **依赖**: 无(Python标准库)
- **性能**: ⭐⭐⭐⭐⭐
- **推荐度**: ⭐⭐⭐⭐⭐

#### 4.1.2 核心功能

```python
class TextLoader:
    def __init__(self):
        self.name = "text"
        self.supported_formats = [".txt", ".md", ".markdown"]
    
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取纯文本内容"""
        try:
            # 尝试多种编码
            encodings = ['utf-8', 'gbk', 'gb2312', 'iso-8859-1']
            text = None
            
            for encoding in encodings:
                try:
                    with open(file_path, 'r', encoding=encoding) as f:
                        text = f.read()
                    break
                except UnicodeDecodeError:
                    continue
            
            if not text:
                raise ValueError("无法解码文件")
            
            # 按段落分页(每10段一页)
            paragraphs = [p for p in text.split('\n\n') if p.strip()]
            pages = []
            page_size = 10
            
            for i in range(0, len(paragraphs), page_size):
                page_paras = paragraphs[i:i+page_size]
                page_text = "\n\n".join(page_paras)
                pages.append({
                    "page_number": len(pages) + 1,
                    "text": page_text,
                    "char_count": len(page_text)
                })
            
            return {
                "success": True,
                "loader": "text",
                "pages": pages,
                "full_text": text,
                "total_pages": len(pages),
                "total_chars": len(text)
            }
            
        except Exception as e:
            return {
                "success": False,
                "loader": "text",
                "error": str(e)
            }
```

#### 4.1.3 编码检测

支持多种编码自动检测:
- UTF-8 (优先)
- GBK (中文)
- GB2312 (简体中文)
- ISO-8859-1 (西文)

#### 4.1.4 使用场景

✅ **适合**:
- 纯文本文件
- Markdown文档
- 代码文件
- 日志文件

## 5. 加载器选择指南

### 5.1 决策树

```
文档类型?
├─ PDF
│  ├─ 标准PDF文档? → PyMuPDF (推荐)
│  ├─ 简单PDF? → PyPDF
│  └─ 复杂布局/表格? → Unstructured
├─ DOCX
│  └─ 使用 DOCX Loader
├─ DOC
│  └─ 使用 DOC Loader (多方法降级)
└─ TXT/MD
   └─ 使用 Text Loader
```

### 5.2 性能对比

| 加载器 | 1MB文件 | 10MB文件 | 50MB文件 |
|-------|---------|----------|----------|
| PyMuPDF | ~1s | ~5s | ~20s |
| PyPDF | ~2s | ~10s | ~40s |
| Unstructured | ~5s | ~30s | ~120s |
| DOCX | ~0.5s | ~3s | ~15s |
| DOC | ~2s | ~10s | ~50s |
| Text | ~0.1s | ~0.5s | ~2s |

### 5.3 质量评估指标

1. **文本准确性**: 提取文本的完整性和准确度
2. **元数据完整性**: 提取的元信息是否齐全
3. **结构保持**: 是否保留原文档结构
4. **特殊内容**: 表格、图片等特殊内容的处理

## 6. 最佳实践

### 6.1 性能优化
1. 优先使用原生加载器(PyMuPDF, python-docx)
2. 大文件考虑分页加载
3. 缓存加载结果避免重复处理
4. 使用异步处理提升响应速度

### 6.2 错误处理
1. 实现多层级降级策略
2. 提供详细的错误信息
3. 记录加载失败日志
4. 支持手动重试

### 6.3 扩展开发

2. 返回标准化的数据格式
