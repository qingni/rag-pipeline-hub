## 1. 加载器概览

### 1.1 加载器对比表

| 加载器 | 支持格式 | 优点 | 缺点 | 适用场景 | 性能 |
|-------|---------|------|------|---------|------|
| **Docling Serve** | PDF/DOCX/XLSX/PPTX/HTML/图片 | HTTP服务、OCR支持、表格识别 | 需独立部署 | 复杂文档(推荐) | ⭐⭐⭐⭐ |
| **PyMuPDF** | PDF | 快速、准确、功能全 | 依赖C库 | 一般PDF文档 | ⭐⭐⭐⭐⭐ |
| **PyPDF** | PDF | 纯Python、轻量 | 功能有限 | 简单PDF | ⭐⭐⭐⭐ |
| **Unstructured** | PDF等 | 支持复杂布局 | 速度慢 | 复杂文档 | ⭐⭐⭐ |
| **DOCX** | DOCX | 官方支持、准确 | 仅DOCX | Word文档 | ⭐⭐⭐⭐⭐ |
| **DOC** | DOC | 支持旧格式 | 需外部工具 | 旧版Word | ⭐⭐⭐ |
| **HTML** | HTML | 增强图片提取、多格式输出 | - | 网页文档 | ⭐⭐⭐⭐⭐ |
| **JSON** | JSON | 结构化文本提取、表格化 | - | 配置/数据文件 | ⭐⭐⭐⭐⭐ |
| **Text** | TXT/MD | 极快、无依赖 | 仅纯文本 | 文本文件 | ⭐⭐⭐⭐⭐ |

## 2. Docling Serve 主解析器

### 2.1 基本信息
- **依赖**: `httpx` (HTTP客户端)
- **服务**: Docling Serve（独立部署）
- **默认地址**: http://localhost:5001
- **性能**: ⭐⭐⭐⭐
- **推荐度**: ⭐⭐⭐⭐⭐ (复杂文档首选)

### 2.2 核心特性

**HTTP 服务模式**:
- 独立部署，避免 Backend 启动时的长时间初始化
- 支持同步/异步两种调用模式
- 可用性缓存，自动重连

**OCR 支持**:
- 内置 RapidOCR 引擎
- 支持中文（简体）和英文识别
- 强制 OCR 模式，处理扫描件

**多格式支持**:
- PDF、DOCX、DOC、PPTX、PPT
- XLSX、XLS、HTML、Markdown
- 图片格式（PNG、JPG、TIFF、BMP、GIF）

```python
class PyMuPDFLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取PDF文本内容"""
        doc = fitz.open(file_path)
        
        # 提取元数据
        metadata = {
            "page_count": len(doc),
            "title": doc.metadata.get("title", ""),
            "author": doc.metadata.get("author", ""),
            "subject": doc.metadata.get("subject", ""),
            "creator": doc.metadata.get("creator", ""),
            "producer": doc.metadata.get("producer", ""),
            "format": doc.metadata.get("format", "")
        }
        
        # 逐页提取文本
        pages = []
        for page_num in range(len(doc)):
            page = doc[page_num]
            text = page.get_text()
            
            pages.append({
                "page_number": page_num + 1,
                "text": text,
                "char_count": len(text)
            })
        
        return {
            "success": True,
            "loader": "pymupdf",
            "metadata": metadata,
            "pages": pages,
            "full_text": "\n\n".join([p["text"] for p in pages]),
            "total_pages": len(pages),
            "total_chars": sum(p["char_count"] for p in pages)
        }
```

```python
class UnstructuredLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """使用Unstructured提取文本"""
        from unstructured.partition.auto import partition
        
        elements = partition(filename=file_path)
        
        # 组织为页面结构
        pages = []
        current_page = {"page_number": 1, "text": "", "char_count": 0}
        
        for element in elements:
            text = str(element)
            current_page["text"] += text + "\n"
            current_page["char_count"] += len(text)
            
            # 简单分页逻辑(可根据element类型优化)
            if len(current_page["text"]) > 2000:
                pages.append(current_page)
                current_page = {
                    "page_number": len(pages) + 1,
                    "text": "",
                    "char_count": 0
                }
        
        if current_page["text"]:
            pages.append(current_page)
        
        return {
            "success": True,
            "loader": "unstructured",
            "pages": pages,
            "total_pages": len(pages),
            "total_chars": sum(p["char_count"] for p in pages)
        }
```

```python
class DocxLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取DOCX文本"""
        from docx import Document
        
        doc = Document(file_path)
        
        # 提取段落
        paragraphs = []
        full_text = []
        
        for para in doc.paragraphs:
            if para.text.strip():
                paragraphs.append(para.text)
                full_text.append(para.text)
        
        # 组织为"页面"(每5段一页)
        pages = []
        page_size = 5
        for i in range(0, len(paragraphs), page_size):
            page_paras = paragraphs[i:i+page_size]
            page_text = "\n\n".join(page_paras)
            pages.append({
                "page_number": len(pages) + 1,
                "text": page_text,
                "char_count": len(page_text)
            })
        
        return {
            "success": True,
            "loader": "docx",
            "pages": pages,
            "full_text": "\n\n".join(full_text),
            "total_pages": len(pages),
            "total_chars": sum(p["char_count"] for p in pages)
        }
```

#### 3.1.3 使用场景

✅ **适合**:

```python
def extract_tables(self, file_path: str) -> List[List[List[str]]]:
    """提取表格数据"""
    doc = Document(file_path)
    tables = []
    
    for table in doc.tables:
        table_data = []
        for row in table.rows:
            row_data = [cell.text for cell in row.cells]
```

```python
class DocLoader:
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """尝试多种方法提取DOC文本"""
        
        # 方法1: antiword (最准确)
        text = self._extract_with_antiword(file_path)
        if text:
            return self._format_result(text, "antiword")
        
        # 方法2: python-docx (部分支持)
        text = self._extract_with_docx(file_path)
        if text:
            return self._format_result(text, "docx-fallback")
        
        # 方法3: textract (最后手段)
        text = self._extract_with_textract(file_path)
        if text:
            return self._format_result(text, "textract")
        
        return {
            "success": False,
            "loader": "doc",
            "error": "无法提取DOC文件内容"
        }
    
    def _extract_with_antiword(self, file_path: str) -> str:
        """使用antiword命令行工具"""
        try:
            import subprocess
            result = subprocess.run(
                ['antiword', file_path],
                capture_output=True,
                text=True,
                timeout=30
            )
            return result.stdout if result.returncode == 0 else None
        except Exception:
            return None
```

#### 4.1.2 核心特性

**多格式输出**:
- `full_text` (Markdown): 包含 `![alt](src)` 格式，支持前端图片预览
- `structured_text` (RAG优化): 包含 `[IMAGE_0: ...]` 标记，便于检索

**增强图片提取**:
- 多源 URL 支持（src、data-src、data-lazy-src、srcset）
- 完整元数据（尺寸、类名、ID、loading策略）
- 智能标题查找（figcaption、title、aria-label、aria-describedby）
- 上下文信息（父元素、相邻文本各100字符）

#### 4.1.3 核心功能

```python
class HTMLLoader:
```

```python
def _find_image_caption(self, img) -> Optional[str]:
    """智能查找图片标题"""
    # 1. 检查父级 figure 的 figcaption
    figure = img.find_parent('figure')
    if figure:
        figcaption = figure.find('figcaption')
        if figcaption:
            return figcaption.get_text(strip=True)
    
    # 2. 检查 title 属性
    if img.get('title'):
        return img.get('title')
    
    # 3. 检查 aria-label
    if img.get('aria-label'):
        return img.get('aria-label')
    
    # 4. 检查 aria-describedby 引用
    describedby_id = img.get('aria-describedby')
    if describedby_id:
        desc_element = soup.find(id=describedby_id)
        if desc_element:
            return desc_element.get_text(strip=True)
    
    return None

def _get_image_context(self, img) -> Dict[str, Any]:
    """获取图片上下文信息"""
    context = {}
    
    # 父元素信息
    parent = img.parent
    if parent:
        context["parent_tag"] = parent.name
        context["parent_class"] = ' '.join(parent.get('class', []))
    
    # 前后相邻文本（各100字符）
    prev_text = self._get_sibling_text(img.previous_siblings, 100)
    next_text = self._get_sibling_text(img.next_siblings, 100)
    
    if prev_text:
        context["text_before"] = prev_text
    if next_text:
        context["text_after"] = next_text
    
    return context
```

### 4.2 JSON Loader（增强版）

#### 4.2.1 基本信息
- **依赖**: 无（Python标准库）
- **性能**: ⭐⭐⭐⭐⭐
- **推荐度**: ⭐⭐⭐⭐⭐

#### 4.2.2 核心特性

- **结构化文本提取**: 递归提取所有字符串值，保留键值关系
- **表格化输出**: 自动识别对象数组，转换为表格结构
- **结构类型识别**: 区分 object、array、array_of_objects、primitive
- **原始数据保留**: metadata 中保留完整 JSON 结构

#### 4.2.3 核心功能

```python
class JSONLoader:
    """Load JSON documents."""
    
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取 JSON 文档内容"""
        with open(file_path, 'r', encoding='utf-8') as f:
            data = json.load(f)
        
        # 提取文本内容（用于 RAG 检索）
        extracted_text = self._extract_text_content(data)
        
        # 如果提取到内容则使用，否则回退到格式化 JSON
        full_text = extracted_text if extracted_text else json.dumps(data, indent=2, ensure_ascii=False)
        
        # 提取表格（如果数据是对象数组）
        tables = self._extract_tables(data)
        
        return {
            "success": True,
            "loader": self.name,
            "metadata": {
                "format": "json",
                "structure_type": self._get_structure_type(data),
                "raw_json": data  # 保留原始 JSON 结构
            },
            "pages": [{"page_number": 1, "text": full_text, ...}],
            "tables": tables,
            "full_text": full_text
        }
    
    def _extract_text_content(self, data: Any, prefix: str = "") -> str:
        """递归提取文本内容"""
        text_parts = []
        
        if isinstance(data, dict):
            for key, value in data.items():
                if isinstance(value, str):
                    text_parts.append(f"{key}: {value}")
                elif isinstance(value, (dict, list)):
                    nested = self._extract_text_content(value, f"{prefix}{key}.")
                    if nested:
                        text_parts.append(nested)
        elif isinstance(data, list):
            for i, item in enumerate(data):
                if isinstance(item, str):
                    text_parts.append(item)
                elif isinstance(item, (dict, list)):
                    nested = self._extract_text_content(item, f"{prefix}[{i}].")
                    if nested:
                        text_parts.append(nested)
        
        return "\n".join(text_parts)
    
    def _extract_tables(self, data: Any) -> list:
        """从 JSON 中提取表格结构"""
        tables = []
        
        # 如果数据是对象数组，转换为表格
        if isinstance(data, list) and len(data) > 0 and isinstance(data[0], dict):
            headers = list(data[0].keys())
            rows = [[str(item.get(h, "")) for h in headers] for item in data if isinstance(item, dict)]
            
            tables.append({
                "page_number": 1,
                "table_index": 0,
                "headers": headers,
                "rows": rows,
                "row_count": len(rows),
                "col_count": len(headers)
            })
        
        return tables
    
    def _get_structure_type(self, data: Any) -> str:
        """判断 JSON 结构类型"""
        if isinstance(data, dict):
            return "object"
        elif isinstance(data, list):
            if len(data) > 0 and isinstance(data[0], dict):
                return "array_of_objects"
            return "array"
        return "primitive"
```

```python
class TextLoader:
    def __init__(self):
        self.name = "text"
        self.supported_formats = [".txt", ".md", ".markdown"]
    
    def extract_text(self, file_path: str) -> Dict[str, Any]:
        """提取纯文本内容"""
        try:
            # 尝试多种编码
            encodings = ['utf-8', 'gbk', 'gb2312', 'iso-8859-1']
            text = None
            
            for encoding in encodings:
                try:
                    with open(file_path, 'r', encoding=encoding) as f:
                        text = f.read()
                    break
                except UnicodeDecodeError:
                    continue
            
            if not text:
                raise ValueError("无法解码文件")
            
            # 按段落分页(每10段一页)
            paragraphs = [p for p in text.split('\n\n') if p.strip()]
            pages = []
            page_size = 10
            
            for i in range(0, len(paragraphs), page_size):
                page_paras = paragraphs[i:i+page_size]
                page_text = "\n\n".join(page_paras)
                pages.append({
                    "page_number": len(pages) + 1,
                    "text": page_text,
                    "char_count": len(page_text)
                })
            
            return {
                "success": True,
                "loader": "text",
                "pages": pages,
                "full_text": text,
                "total_pages": len(pages),
                "total_chars": len(text)
            }
            
        except Exception as e:
            return {
                "success": False,
                "loader": "text",
                "error": str(e)
            }
```
