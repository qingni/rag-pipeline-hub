# 向量化历史记录和结果卡片功能

## 概述

参考文件分块功能的布局设计，为文档向量化页面添加了：
1. **文档向量结果卡片** - 显示当前文档最新的向量化结果摘要
2. **历史向量列表卡片** - 显示和管理所有向量化历史记录

## 新增组件

### 1. EmbeddingResultCard.vue
**位置**: `frontend/src/components/embedding/EmbeddingResultCard.vue`

**功能**:
- 显示最新向量化结果的摘要信息
- 文档信息（文档名、模型名）
- 统计数据（向量维度、成功/失败数量、处理时间）
- 状态指示器（成功/部分成功/失败）
- 快捷操作按钮（查看详情、导出向量）

**UI元素**:
- 卡片头部：标题 + 状态标签 + 时间戳
- 文档信息区：文档名称、向量模型
- 统计网格：4列展示关键指标
- 错误提示：失败时显示错误信息
- 操作按钮：查看详情、导出向量

### 2. EmbeddingHistoryList.vue
**位置**: `frontend/src/components/embedding/EmbeddingHistoryList.vue`

**功能**:
- 列表展示所有向量化历史记录
- 搜索过滤（按文档名）
- 状态过滤（成功/部分成功/失败）
- 分页显示（每页10条）
- 单条记录操作（查看、导出、删除）

**UI元素**:
- 卡片头部：标题 + 刷新按钮
- 过滤区域：搜索框 + 状态下拉选择
- 历史列表：
  - 文档名称 + 状态标签
  - 元数据（模型、维度、成功数、时间）
  - 操作按钮（查看、导出、删除）
- 分页控件

### 3. DocumentEmbedding.vue 布局改造
**位置**: `frontend/src/views/DocumentEmbedding.vue`

**改造内容**:
- 采用 `t-layout` + `t-tabs` 布局（参考分块功能）
- 左侧：控制面板（380px固定宽度）
- 右侧：Tab标签页
  - Tab 1: **向量化结果** - 双栏布局
    - 左栏：文档向量结果卡片
    - 右栏：向量详情展示
  - Tab 2: **历史记录** - 历史列表

## Store 扩展

### embedding.js 新增状态和方法

**新增状态**:
```javascript
embeddingHistory: ref([])  // 历史记录列表
```

**新增方法**:
1. `fetchEmbeddingHistory(documentId)` - 获取历史记录列表
2. `fetchEmbeddingResultById(resultId)` - 根据ID获取结果
3. `deleteEmbeddingResult(resultId)` - 删除向量化结果

## Service 扩展

### embeddingService.js 新增方法

**新增方法**:
```javascript
deleteResult(resultId)  // 删除向量化结果API
```

现有方法（复用）:
- `listResults(params)` - 列出向量化结果
- `getResultById(resultId)` - 获取单个结果

## 功能流程

### 1. 查看最新结果
```
用户选择文档
  ↓
自动加载最新向量化结果
  ↓
左侧卡片显示摘要
  ↓
右侧显示详细向量数据
```

### 2. 查看历史记录
```
切换到"历史记录"Tab
  ↓
加载所有历史记录
  ↓
可搜索、过滤、分页
  ↓
点击"查看"按钮
  ↓
切换回"向量化结果"Tab显示详情
```

### 3. 删除记录
```
在历史列表中点击"删除"
  ↓
确认对话框
  ↓
调用删除API
  ↓
从列表中移除
  ↓
如果是当前显示的结果，清空显示
```

## 样式设计

### 配色方案
- 主色：`#3b82f6` (蓝色)
- 成功：`#10b981` (绿色)
- 警告：`#f59e0b` (橙色)
- 错误：`#ef4444` (红色)
- 背景：`#f9fafb` (浅灰)

### 卡片样式
- 白色背景 + 浅灰边框
- 圆角：8px
- 内边距：20px 24px
- 悬停效果：蓝色边框 + 阴影

### 响应式布局
- 最小宽度：1280px
- 左侧固定：380px
- 右侧自适应
- 双栏各占50%

## 用户体验优化

1. **自动加载**: 选择文档后自动加载最新结果
2. **智能切换**: 
   - 向量化完成后自动切换到结果Tab
   - 查看历史记录后自动切换到结果Tab
3. **状态提示**: 清晰的成功/失败状态展示
4. **快捷操作**: 
   - 卡片级别的快捷按钮
   - 列表项的行内操作
5. **数据持久**: 删除操作有确认提示
6. **性能优化**: 分页加载，避免一次加载过多数据
